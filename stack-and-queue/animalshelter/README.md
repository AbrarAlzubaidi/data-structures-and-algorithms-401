## Approach & Efficiency
1. enqueue: O(1)
2. dequeue: O(1)

## Solution
1. enqueue: 

        shelter.enqueue('cat1', 'cat')
        actual=shelter.dequeue('cat')
        expected='cat called cat1 adopted'
        assert actual == expected

2. dequeue: 

        shelter.enqueue('cat1', 'cat')
        actual=shelter.dequeue('cat')
        expected='cat called cat1 adopted'
        assert actual == expected
